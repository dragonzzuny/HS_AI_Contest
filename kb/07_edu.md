# 도메인: 보육·교육 (edu)

### [c-edu-01] 어린이집 입소 신청 (외국인·다문화 자녀)
- 분야: 보육교육
- 대상: 만 0~5세 외국인·다문화가정 영유아 보호자
- 질문예시: "외국인인데 아이 어린이집 어떻게 신청해요?" / "How can a foreigner apply for daycare (eorinijip) enrollment?"
- 답변요지(한국어): 1. 아이사랑 포털(www.childcare.go.kr) 가입 후 아동 등록, 공동인증서 등록 → '어린이집 > 입소대기' 신청, 또는 어린이집 직접 방문 신청. 2. 다문화가족 자녀는 입소 우선순위 1순위에 해당. 3. 재원 중이면 2곳, 미재원이면 3곳까지 대기 신청 가능.
- 필요서류: 주민등록등본(결혼이민자·귀화자 기재) 또는 가족관계증명서, 외국인등록증 사본(없으면 외국인등록사실증명서)
- 담당기관/부서: 화성시 영유아보육과 / 각 구 돌봄·가정보육과
- 연락처: ☎ 영유아보육과 031-5189-6084
- 위치/방문: 거주지 관할 어린이집 또는 행정복지센터
- 출처: [아이사랑 입소대기](https://childcare.go.kr/cpin/contents/040201000000.jsp) · [화성시청 육아지원](https://www.hscity.go.kr/www/partInfo/femaleFamily/Welfare1/Welfare1_3.jsp) — 확인일 2026-06-23
- 신뢰도: 확인됨

### [c-edu-02] 외국인 아동 보육료 지원 (화성시)
- 분야: 보육교육
- 대상: 화성시 관내 어린이집 재원 외국인 영유아(만 0~5세), 보호자·아동 모두 화성시 90일 초과 거주
- 질문예시: "외국인 아이도 보육료 지원받을 수 있나요?" / "Can foreign children get childcare fee support in Hwaseong?"
- 답변요지(한국어): 1. 화성시 거주 외국인 영유아의 어린이집 부모보육료를 국민행복카드 바우처로 지원. 2. 신청기간 2026.1.1.~12.31. 3. 재원 중인 어린이집에 신청서 제출.
- 필요서류: 외국인 아동 보육료 지원 신청서, 보호자·자녀 외국인등록증 사본(앞뒤), 외국인등록 사실증명서
- 담당기관/부서: 화성시 영유아보육과 / 각 구 돌봄·가정보육과
- 연락처: ☎ 영유아보육과 031-5189-6084 / 만세구 돌봄복지과 031-5189-1187 / 효행구 031-5189-7665 / 병점구 031-5189-4153 / 동탄구 가정보육과 031-5189-5205
- 위치/방문: 재원 어린이집
- 출처: [화성시청 육아지원](https://www.hscity.go.kr/www/partInfo/femaleFamily/Welfare1/Welfare1_3.jsp) — 확인일 2026-06-23
- 신뢰도: 확인됨

### [c-edu-03] 초등학교 입학·편입학 (외국인 자녀 학교 배정)
- 분야: 보육교육
- 대상: 취학연령 외국인·이주배경·중도입국 자녀 보호자
- 질문예시: "외국인 아이를 초등학교에 보내려면 어떻게 해요?" / "How do I enroll my foreign child in elementary school?"
- 답변요지(한국어): 1. 외국인가정 자녀는 취학통지서가 발급되지 않으므로, 거주지 인근 초등학교에 직접 방문해 입학(편입학) 상담. 2. 거주지 관할 행정복지센터에서 (재)취학 학교 문의 후 해당 학교에 신청. 3. 학력 증빙서류가 외국어인 경우 원본+번역본 공증 제출, 학력 증빙이 어려우면 학교·교육청이 학년 배정.
- 필요서류: 거주사실 확인서류(주택임대차계약서 등), 출생연월일 확인 공문서(여권 사본 등), 학력 증빙서류(있을 경우, 외국어는 공증 번역본)
- 담당기관/부서: 거주지 인근 초등학교 / 경기도교육청·관할 교육지원청 다문화교육 담당
- 연락처: ☎ 다누리콜센터 1577-1366 (다국어 통번역·상담)
- 위치/방문: 거주지 인근 초등학교, 관할 행정복지센터
- 출처: [경기도교육청 다문화교육 공교육 진입 One-Stop](https://family.gen.go.kr/sub/page.php?page_code=guide_03_01) — 확인일 2026-06-23
- 신뢰도: 부분확인 (화성 관할 교육지원청 직통번호 TODO검증)

### [c-edu-04] 다문화가족 자녀 교육활동비 지원
- 분야: 보육교육
- 대상: 기준중위소득 100% 이하 다문화가족의 만 7~18세 자녀(교육급여 수급자 제외)
- 질문예시: "다문화 가정인데 자녀 교육비 지원이 있나요?" / "Is there education cost support for multicultural family children?"
- 답변요지(한국어): 1. 여성가족부 사업, 연간 지원금액 초등생 40만원·중학생 50만원·고등학생 60만원. 2. 신청자 명의 NH농협카드(채움) 포인트로 지급. 3. 2026년 신청기간 6.1.~6.30., 주소지 가족센터(다문화가족지원센터)에 개별 신청.
- 필요서류: 신청서, 다문화가족 입증서류, 자녀 재학·소득 증빙(신청기관 안내에 따름)
- 담당기관/부서: 거주지 가족센터 / 여성가족부 다문화가족과
- 연락처: ☎ 화성시 가족센터 031-352-1214 / 여성가족부 다문화가족과 02-2100-6369
- 위치/방문: 주소지 관할 가족센터
- 출처: [여성가족부 보도자료](https://www.mogef.go.kr/nw/rpd/nw_rpd_s001d.do?mid=news405&bbtSn=709908) · [복지로 교육활동비 지원](https://www.bokjiro.go.kr/ssis-tbu/twataa/wlfareInfo/moveTWAT52011M.do?wlfareInfoId=WLF00005448) — 확인일 2026-06-23
- 신뢰도: 확인됨 (화성시 자체 추가지원 여부 미확인)
