# 화성ON 지식베이스(KB) 표준 스키마

> 다국어 외국인 생활민원 AI 안내사의 RAG 근거 데이터.
> **원칙: 공식 출처에서만 수집. 미확인 정보는 지어내지 말고 `신뢰도: TODO검증`으로 표시.**

각 민원 항목은 아래 형식의 markdown 블록으로 작성한다. 청크 ID(`c-도메인-번호`)는 인용 추적에 사용된다.

```
### [c-stay-01] 외국인등록증 발급
- 분야: 체류 / 노무 / 건강보험 / 폐기물 / 보육교육 / 운전면허 / 행정 / 지원기관
- 대상: 90일 이상 체류하는 외국인
- 질문예시: "외국인등록 어떻게 해요?" / "How do I register as a foreigner?"
- 답변요지(한국어): 
  1. (단계1)
  2. (단계2)
- 필요서류: 여권, 사진, 수수료 등
- 담당기관/부서: 수원출입국·외국인사무소 (화성 관할) / 화성시 ○○과
- 연락처: ☎ 000-0000-0000
- 위치/방문: 주소 또는 온라인 처리(Hi Korea 등)
- 출처: [기관명 페이지 제목](https://...) — 확인일 2026-06-23
- 신뢰도: 확인됨 | 부분확인 | TODO검증
```

## 도메인 목록 (목표 25~30건)
1. **stay** 체류·외국인등록 (등록증, 체류연장, 주소변경, 재입국)
2. **labor** 노무·임금 (근로계약, 임금체불, 산재, 외국인노동자 상담)
3. **health** 건강보험·의료 (지역가입 의무, 보험료, 병원·응급)
4. **waste** 생활폐기물 (분리배출, 대형폐기물 스티커, 음식물)
5. **edu** 보육·교육 (어린이집·유치원, 초등 입학, 다문화 지원)
6. **license** 운전면허 (외국면허 교환, 면허시험 다국어)
7. **admin** 일반행정 (전입신고, 주민센터, 통번역)
8. **support** 지원기관 (화성시외국인복지센터/글로벌센터 연락처·서비스)

## 6개 지원 언어
한국어(ko) · 영어(en) · 중국어(zh) · 베트남어(vi) · 태국어(th) · 캄보디아어(km)
> 답변은 사용자 언어로, **근거(출처)는 한국어 원문 유지**.
