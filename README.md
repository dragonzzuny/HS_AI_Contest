# 화성ON — 다국어 외국인 생활민원 AI 안내사

> 화성시 거주 외국인이 **자기 나라 말로** 생활민원을 묻고, **근거(자치법규·민원 FAQ 출처)**와 함께
> 정확한 절차·담당부서·연락처를 안내받으며, **신청 서류 초안**까지 받을 수 있는 인용기반 RAG 챗봇.

2026 화성시 시민·공무원 AI 공모전 출품작.

## 배경

화성시는 외국인주민이 전국 최상위권(약 8.1만 명, 양감면은 인구의 38%)인 도시입니다.
그러나 기존 정부·지자체 행정 챗봇은 대부분 한국어 중심이라, 한국어가 익숙하지 않은
외국인 주민은 "어떤 서류가 필요한지, 어디로 가야 하는지"조차 알기 어렵습니다.

**화성ON**은 이 공백을 메웁니다 — 6개 언어(한국어·영어·중국어·베트남어·태국어·캄보디아어)로
질문을 받아, 화성시 공식 자료에 근거해 답하고, 근거 출처를 함께 표시합니다.

## 주요 기능

- **다국어 질의·응답**: 사용자 언어로 묻고 답변. 근거(출처)는 한국어 원문 유지.
- **인용기반 RAG**: 모든 답변에 근거 항목 ID를 인라인 표기 → 클릭하면 원문·출처·담당부서 표시.
  근거에 없으면 단정하지 않고 담당기관 문의로 안내(환각·오답 방지).
- **2계층 지식베이스**:
  - **L1** 외국인 고빈도 생활민원 FAQ 37건(체류·노무·건강보험·폐기물·보육·운전면허·행정·지원기관) — 사람 검증.
  - **L2** 화성시 자치법규(조례·규칙) 839건 → 11,268개 조문 청크(법제처 국가법령정보 Open API 전수 수집).
- **처리 도우미(Action)**:
  - 맞춤 **필요서류 체크리스트**
  - **신청서·진정서 초안 자동 작성**(임금체불 진정서, 대형폐기물 신고) — 빈칸 채우면 한국어 공식 서식 생성
  - 정부24·하이코리아·노동포털 등 정확한 **신청 딥링크**

## 아키텍처

```
[웹앱(브라우저)]  채팅 + 근거 소스패널 + 6개 언어 토글 + 처리 도우미
       │
[FastAPI 단일 앱]
       │  ① 질의 검색(하이브리드)  ② 근거 그라운딩 생성  ③ 처리보조(서류초안·딥링크)
       ▼
[검색층] BGE-M3 임베딩 + Weaviate 하이브리드(키워드+벡터) + 리랭커
[생성층] 로컬 LLM(Ollama) — 다국어 그라운딩 답변
[지식]   L1 FAQ(kb/*.md) + L2 자치법규(법제처 Open API)
```

> LLM/벡터DB가 없어도 **MOCK 모드(키워드 검색 + 템플릿)**로 즉시 작동하며,
> Ollama·Weaviate가 연결되면 실동작(교차언어 임베딩·실답변)으로 자동 승격됩니다.

## 빠른 시작

```bash
pip install -r backend/requirements.txt
python scripts/build_kb.py                 # kb/*.md → kb/build/kb.json
python -m uvicorn backend.app:app --host 0.0.0.0 --port 5200
# 브라우저: http://localhost:5200
```

자치법규(L2) 수집·실동작(Ollama/Weaviate) 전환 등 상세 절차는 **[RUN.md](RUN.md)** 참고.

## 기술 스택

- 백엔드: Python / FastAPI
- 프론트: Vanilla JS (빌드 불필요, 단일 앱이 서빙)
- 임베딩: BGE-M3 (다국어, 무료)
- 벡터 DB: Weaviate (하이브리드 검색)
- 생성 LLM: Ollama (로컬)
- 데이터: 법제처 국가법령정보 Open API, 화성시청·정부24·국민건강보험공단 등 공식 출처

## 디렉터리 구조

```
backend/      FastAPI 앱(검색·생성·처리보조·인용)
web/          프론트(채팅·소스패널·다국어·처리 도우미)
kb/           L1 FAQ(8개 도메인) + 스키마/DB설계/검증목록
scripts/      build_kb / collect_ordinances / ingest_weaviate
docker-compose.weaviate.yml
RUN.md        실행·배포 가이드
```

## 데이터 출처

자치법규는 법제처 국가법령정보 공동활용(open.law.go.kr) Open API로 수집했으며,
민원 FAQ는 화성시청·정부24·하이코리아·국민건강보험공단·고용노동부·도로교통공단 등
공식 출처를 근거로 정리했습니다. 각 항목에 출처 링크와 확인일·신뢰도를 표기합니다.

## 유의

본 서비스의 안내는 참고용이며, 최종 확인은 각 담당기관에 문의해야 합니다.
민감 정보(개인정보·인증정보)는 수집·처리하지 않습니다.
